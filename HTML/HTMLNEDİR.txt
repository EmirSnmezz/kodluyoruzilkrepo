HTML NEDİR ? 

HTML yani Hiper Metin İşaretleme Dili,web sayfalarında gördüğümüz iskelet yapısını oluşturmak için kullanılan metin işaretleme dilidir.Bir progralama dili olarak tanımlanmayan HTML'e metin işaretleme dili denmesinin sebebi,HTML kodlarıyla çalışan bir program yazamadığımızdan kaynaklanmaktadır.


HTML NE YAPAR ?

Web sayfasında gördüğümüz görseller,yazılar ve videoların konumlandırılmasını sağlamaktadır.

Konumlandırılan web sayfalarının web tarayıcılar tarafından doğru bir şekilde görüntülenmesini gerekli kuralları belirler.

HTML aslında bilgisayarda sürekli olarak kullanıdığımız web sitelerinin oluşmasında kullanılan bir dildir.Yazılan kodlar web tarayıcılar tarafından web sayfalarına dönüştürülmektedir.Tabi ki bu web sayfaları sadece HTML  ile oluşmuyor,HTML kodlarının yanında CSS ve JavaScript dilleri de kullanılmakta.
